# Teslo Shop

## Environment Variables
1. Clone the .env.template file and rename it to .env.
2. Change environment variables.

## Lift the database
```bash
$ docker-compose up -d
```

## Installation

```bash
$ npm install
# or
$ yarn install
# or
$ pnpm install
```

## Running the app

```bash
# development
$ npm run dev
# or
$ yarn dev
# or
$ pnpm dev
```
